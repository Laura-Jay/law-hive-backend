# lawHive App Backend 

## Install

`yarn`

## DB Setup Local

Create .env and set `DATABASE_URL` and `PORT` to your liking.

## Running locally

`yarn start:dev`

This will set the env var LOCAL to true, which will cause the db connection configuration to NOT use SSL.
